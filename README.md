# Aswathy & Arun Wedding Website

A beautiful, interactive wedding website for Aswathy & Arun's wedding on 23rd May 2026.

## ✨ Features

- 🎨 **Elegant Design**: Maroon and cream color scheme with natural, realistic styling
- 🖼️ **Split Hero Layout**: Image on left, names and date on right
- 📸 **Photo Gallery**: 4-row horizontal scrolling carousel with 19 wedding photos
  - Traditional photos (aa series) and Western photos (s series) separated
  - Lightbox functionality for full-screen viewing
  - Smooth hover animations and effects
- 🎬 **Video Section**: Two wedding videos with beautiful play controls
- 🎵 **Background Music**: Floating music player with wedding song
- 💒 **Wedding Details**: Complete venue information with map links
  - Thalikettu at Guruvayur Sree Krishna Temple
  - Wedding celebration at Munnas Auditorium
  - Reception at Hayath Auditorium
- 👰🤵 **Bride & Groom Cards**: Detailed family information
- ✨ **Smooth Animations**: AOS library for scroll-based animations
- 📱 **Fully Responsive**: Perfect on mobile, tablet, and desktop
- ⌨️ **Keyboard Navigation**: Arrow keys, space, home/end support
- 🎯 **Interactive Elements**: Parallax effects and cursor trails

## 🎨 Design Elements

- **Color Scheme**: Maroon (#7d2e2e) and warm cream tones
- **Typography**: Crimson Text for names, Lato for body text
- **Layout**: Clean, natural design without gradients
- **Images**: Optimized 260px × 380px photo sizes
- **Animations**: Elegant hover effects and transitions

## 🚀 How to Use

### Option 1: Open Locally
1. Download all files to a folder
2. Open `index.html` in any modern web browser
3. Enjoy the full wedding website experience

### Option 2: Host Online (Recommended for Sharing)

#### GitHub Pages (Free):
1. Create account at [github.com](https://github.com)
2. Create new repository named `aswathy-arun-wedding`
3. Upload ALL files (including videos and music)
4. Go to Settings → Pages → Deploy from main branch
5. Share your link: `https://yourusername.github.io/aswathy-arun-wedding`

#### Netlify (Easiest):
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop your entire folder
3. Get instant link to share with guests

## 🛠️ Customization

### Change Photos:
1. Replace image files with same names
2. Or update filenames in `index.html`

### Update Content:
- Edit names, dates, venues in `index.html`
- Modify colors in `styles.css`
- Replace music/video files

### Add Features:
- Additional photos: Copy photo-item divs
- More videos: Duplicate video-item-large sections
- New sections: Follow existing HTML structure
